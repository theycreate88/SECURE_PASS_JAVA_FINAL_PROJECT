# SecurePass

SecurePass is a Java-based security application designed to securely store, manage, and verify user credentials.  
The project focuses on authentication, password security, and safe data handling using best practices in Java.

---

##  Features

- Secure user registration and login  
- Password hashing (no plain-text storage)  
- Authentication validation  
- User-friendly console / GUI-based interaction  
- Modular and clean Java code structure  
- Focus on security principles and best practices  

---

##  Technologies Used

- Java  
- Java Collections  
- File Handling / Database (based on implementation)  
- Cryptographic Hashing  

---

##  How It Works

1. User registers with a username and password  
2. Password is hashed before storage  
3. During login, entered password is hashed and compared  
4. Access is granted only if credentials match  

---
 

##  Security Concepts Used

- Password Hashing  
- Authentication Validation  
- No Plain-text Password Storage  
- Basic Access Control Logic
